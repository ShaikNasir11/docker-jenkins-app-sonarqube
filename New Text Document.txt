# Docker Jenkins + SonarQube + Python App CI/CD Pipeline

This project demonstrates a simple **Dockerized CI/CD pipeline** setup using **Jenkins**, **SonarQube**, and a Python Flask application.

---

## 📦 Project Overview

- Dockerize a sample Python Flask application.
- Set up a Jenkins container for CI/CD automation.
- Integrate SonarQube for code quality analysis.
- Build Docker images, run containers, and perform static code analysis via Jenkins pipeline.
